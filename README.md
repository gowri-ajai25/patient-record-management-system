# Patient Record Management System

## Overview

This project is a desktop-based application developed to manage patient records efficiently in a hospital environment. It provides a structured way to store, update, and retrieve patient information using a graphical user interface.

---

## Problem Statement

Manual handling of patient records in hospitals is time-consuming, error-prone, and difficult to maintain. There is a need for a system that ensures efficient data management and quick access to patient information.

---

## Solution

This system offers a digital solution for managing patient records. It allows healthcare staff to perform operations such as adding, updating, viewing, and deleting patient data in a reliable and organized manner.

---

## Features

* Add new patient records
* View patient details
* Update existing records
* Delete patient records
* Search functionality for patient data
* User-friendly interface using Java Swing
* Database connectivity using JDBC

---

## Tech Stack

### Programming Language

* Java

### GUI

* Java Swing

### Database

* MySQL

### Connectivity

* JDBC (Java Database Connectivity)

---

## System Architecture

The application follows a client-database architecture:

* The user interacts with the system through a Java Swing interface
* The backend logic communicates with the database using JDBC
* Patient data is stored and managed in a MySQL database

---

## Installation and Setup

1. Clone the repository:
   git clone https://github.com/gowri-ajai25/patient-record-management-system.git

2. Open the project in NetBeans or any Java IDE

3. Set up MySQL database:

   * Create a database (e.g., hospital_db)
   * Create required tables for patient records

4. Update database configuration in the code:

   * Database URL
   * Username
   * Password

5. Run the application from the IDE
